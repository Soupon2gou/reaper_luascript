## extend_item.lua 
 アイテムを次のアイテムまで引き伸ばすluaスクリプトです。
 アクションタブからアクションを開いて、そのまま新規アクションからReaScript読み込みよりご使用ください

## kkkanten_Deselect odd-numbered items from selected items.lua : 選択中のトラックの奇数アイテムの選択を外します(実質的には選択)

右クリックとかでインストールを押すと導入できます。
